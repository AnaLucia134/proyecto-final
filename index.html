Preguntas introductorias

¿Qué es un framework?

Es una estructura previa que se puede aprovechar para desarrollar un proyecto, es como una especie de plantilla, un esquema conceptual, que simplifica la elaboración de una tarea, ya que solo es necesario complementarlo de acuerdo a lo que se quiere realizar.

¿Qué es Bootstrap 5 y cuál es su origen?

Bootstrap es el framework más popular para el desarrollo de interfaces web con un diseño moderno, adaptadas a las pantallas de los dispositivos. Su origen se remonta al año 2011. Fué desarrollado por la compañía Twitter y su principal función era la de dar forma a un determinado sitio web a través de las librerías CSS, las cuales incluyen una amplia selección de elementos: botones, cuadros, menús, diferentes tipografías… Tan solo unos meses después de su lanzamiento, Bootstrap fué liberado con la licencia Mit. Fué lanzado el 19 de agosto de 2011 y hasta la actualidad han lanzado más de 20 versiones del Framework, actualmente Bootstrap es de uso libre y también es usado por la compañía Twitter en sus proyectos.

¿Qué es diseño responsive en páginas web?

El diseño web responsive o adaptativo es una técnica de diseño web que busca la correcta visualización de una misma página en distintos dispositivos. Desde ordenadores de escritorio a tablets y móviles. Se trata de redimensionar y colocar los elementos de la web de forma que se adapten al ancho de cada dispositivo permitiendo una correcta visualización y una mejor experiencia de usuario.

¿Cuál es la diferencia entre elemento y componente web?

Los componentes web son bloques de código que encapsulan la estructura interna de elementos HTML, incluyendo CSS y JavaScript, permitiendo así que el código se pueda volver a usar como se quiera en otras webs y aplicaciones. Los elementos son parte de la especificación del estándar de componentes web, junto con las importaciones de DOM DOM, plantillas e HTML. En conclusión, los elementos son partes especificas de los componentes.

¿Qué son las Github pages?

GitHub Pages es una forma de alojar nuestros proyectos en una página web, sin necesidad de tener conocimientos en servidores.

¿Cuál es la importancia de documentar o comentar código?

El propósito de usar comentarios es el poder compensar la falta de expresividad del propio código. Por ello podemos decir que los comentarios son una forma de expresar un fallo como programador al momento de programar o el porqué de algún elemento.

¿Por qué existe la sección de atribución en un sitio web?

Para ser un punto de referencia que sirve para identificar los puntos de contacto que reciben crédito por la generación de conversiones (extraños, visitantes, prospectos, clientes).

¿Cuál es la importancia de la programacion?

Es importante ya que ha sido la causante de que la tecnología haya podido avanzar hasta como la encontramos en la actualidad, permitiendo que se desarrollen inventos que faciliten la vida a todas las personas.

HTML + CSS
HTML son las siglas de HyperText Markup Language, es decir, lenguaje de marcado de hipertexto, que constituye un estándar asumido y reconocido a nivel mundial. Sus normas las establece una entidad sin fines de lucro llamada World Wide Web Consortium (W3C) que lo define como «un lenguaje reconocido universalmente y que permite publicar información de forma global». En cuanto a CSS (del inglés Cascading Style Sheets u “hojas de estilo en cascada”), es el lenguaje de marcado que determina el aspecto con el que se visualiza el contenido de un sitio web. Esto pasa por elementos como el color, estilos de letra o imágenes, entre muchos otros, y su código se conjuga con el de HTML para crear páginas completas.
Maquetado Responsive
Es un diseño web adaptable es el diseño que es capaz de adaptarse a diferentes tipos de dispositivos. El sistema detecta el ancho de la pantalla y a partir de ahí adapta todos los elementos de la página para ofrecer una mejor experiencia al usuario.
Github Pages
GitHub Pages es un espacio gratuito para almacenar los archivos que ejecutan un sitio web y alojar ese sitio para que las personas lo visiten, este está dentro de GitHub que es una plataforma visual para utilizar git, un sistema de versionado o, en otras palabras, de registro de cambios realizados en los archivos.
Bootstrap 5
Bootstrap es uno de los marcos más populares para el desarrollo de sitios web. La herramienta proporciona plantillas para CSS y HTML con el que puede configurar un diseño web moderno de una manera relativamente simple, Bootstrap 5 es la actualización mas reciente del mismo.
Bachillerato en computacion
La carrera de Bachillerato en Computación capacita a los estudiantes para poder desenvólverse en el campo de la computación comercial como operador y programador, aplicando los conocimientos adquiridos a través de la duración de la carrera.
JavaScript
Es un lenguaje de programación o de secuencias de comandos que te permite implementar funciones complejas en páginas web, cada vez que una página web hace algo más que sentarse allí y mostrar información estática para que la veas, muestra oportunas actualizaciones de contenido, mapas interactivos,entre otros.

Reflexión

Esta práctica me ayudo a terminar de integrar todo lo que he aprendido en esta área, por lo general se implementa un solo elemento a la página o varios, pero sin información, esta vez tuve la oportunidad de integrarlos todos y poder experimentar con el código de los mismo, descubrí varias cosas que quizás sin este proyecto no las hubiera encontrado, fue algo bastante interesante experimentar con la modificación de cosas específicas de ciertos elementos. De forma personal, esto me servirá en un futuro ya que me quiero dedicar a la programación web y el poder trabajar con tantos elementos que se pueden implementar de tantas formas es algo que me ayuda a crecer como programadora, todavía tengo mucho que aprender, pero esta fue una buena forma de empezar y saber cuáles fueron mis puntos débiles. Considero que hace falta más implementación de practica con los elementos complicados en las páginas web, ya que solo se profundiza en los básicos y al momento de querer trabajar con unos complicados se nos hace más difícil y tenemos que acudir a medios externos para poder aprender cómo utilizarlos. Los conocimientos que adquirí durante este tiempo fueron bastante básicos y muchos elementos los tuve que aprender de forma autodidacta, no considero que sea un factor negativo ya que me ayuda a desenvolver de mejor manera mi lado de autoaprendizaje y así poder darme una idea de lo que soy capaz de hacer por mi cuenta, ya que en un futuro no necesariamente tendré a alguien que me enseñe las cosas. Lo que he aprendido hasta la fecha fue bastante básico, aunque se podría decir que fue el justo para el grado de conocimiento que tenemos, fue bastante bueno, pero cabe resaltar que, si fue bastante lento, aunque esto es una buena influencia para comprender mejor los temas y así integrarlos al 100%, esto es un factor que nos ayudó a poder desarrollar este proyecto con conocimientos propios y así autoevaluarnos de lo que realmente conocemos. Desde el inicio de la carrera hasta la fecha he aprendido como programas y modificas elemento de páginas web en HTML y CSS, el desarrollo de programas en free pascal, en visual studio code, lógica de programación, desarrollo en diagramas de flujo, y modificaciones básicas en JavaScript.
